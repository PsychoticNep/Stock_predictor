# Stock Predictor with Java

A stock prediction application built with Java and Spring Boot, leveraging REST APIs for real-time market data and machine learning for price prediction.

## 🛠️ Languages & Frameworks

### Language
- **Java** — Primary programming language

### Framework & Build
- **Spring Boot** — Application framework for building the backend REST services
- **Maven** — Build automation and dependency management

### Data & API
- **Alpha Vantage / Yahoo Finance API** — REST APIs for fetching real-time and historical stock data

### Database
- **MySQL** — Relational database for storing stock data and predictions
- **H2** — In-memory database for development and testing
- **Hibernate / JPA** — ORM for database interaction

### Machine Learning (optional)
- **Weka** — Machine learning library for prediction models
- **DL4J (Deeplearning4j)** — Deep learning framework for advanced prediction algorithms
